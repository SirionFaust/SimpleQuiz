Interactive JS Quiz Game

A sleek, responsive, and dynamic Quiz Application built with **Vanilla JavaScript**, **HTML5**, and **CSS3**.

 Features
- **Dynamic Questions:** Questions and answers are loaded from a JavaScript object.
- **Real-time Scoring:** Track your progress and score as you play.
- **Visual Feedback:** Instant color coding for correct (green) and incorrect (red) answers.
- **Progress Bar:** A smooth visual indicator of how far you are in the quiz.
- **Responsive Design:** Works perfectly on desktops, tablets, and mobile phones.

 Technologies Used
- **HTML5:** For the structural layout.
- **CSS3:** For custom styling and animations.
- **JavaScript (ES6+):** For the game logic, DOM manipulation, and event handling.

How to Play
1. Click the **Start Quiz** button to begin.
2. Read the question and select one of the four options.
3. The game will highlight the correct answer and move to the next question automatically after 1 second.
4. View your final score and performance message at the end!
